# 1.4 - Unit Testing Assignment

> Phillip Cantu<br /><pvcantu@student.fullsail.edu><br />Monday, March 4, 2026

This repo is for the 1.4 Unit Testing Assignment in my **Full Sail University Web Application Integration - Online Course** and shows basic unit test coverage on a simple JavaScript function using Node.js and Jest.

## 📹 Video Demo

A screen‑recorded walkthrough.

👉 Watch it on Youtube: [youtube.com/watch?v=4Tlv28EsZ7c](https://www.youtube.com/watch?v=4Tlv28EsZ7c)

## Files

- index.js _main entry point: function logic that's exported_
- total.test.js _jest test file: unit testing logic that imports function_
- package.json _displays only Jest is needed and custom test script to run_

## How to run

_prequisites: Node v20+ required_

1. Run `git clone` to clone the repo to your local machine
2. Run `cd` to change directory into the repo
3. Run `npm install`
4. Run `npm test`
